# SocialFly AI - Architecture Overview

## ğŸ—ï¸ System Architecture

**SocialFlyAI** is a **unified Next.js application** running entirely on **PORT 3000**. It is NOT a microservices architecture - everything runs in a single process:

- **Frontend**: React/Next.js UI with tab-based navigation
- **Backend**: Next.js API Routes (`/api/*`)
- **Services**: TypeScript service classes in `/lib/services/`
- **Port**: All traffic goes through **PORT 3000**
